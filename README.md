# 项目介绍
一款Linux权限维持+后渗透的工具，功能：端口扫描，敏感信息，指纹识别，IP伪装，键盘监控，进程隐藏，内容搜索，文件搜索，下载工具，edr/av识别，权限维持，ssh连接伪装等多个功能。

在针对文件操作的时候，操作之前会copy文件的时间戳，操作完成会恢复到修改之前状态。
